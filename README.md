Data & Code Availability

Los datos procesados, scripts de análisis y figuras utilizadas en este estudio están disponibles públicamente en GitHub:

https://github.com/empedocles-democrito/Structural-validation-of-autonomous-dynamics-in-urban-activation-fields

El repositorio contiene los archivos .csv, scripts .py y figuras .png necesarios para reproducir los diagnósticos empíricos del artículo, incluyendo: construcción del campo de activación, tests de propagación espacial, análisis no-Markoviano, kernels temporales, análisis latente y reconstrucción de operadores efectivos y Hamiltoniano continuo.

Los datos originales de taxis de NYC provienen de fuentes públicas; el repositorio incluye versiones procesadas compatibles con los scripts.
